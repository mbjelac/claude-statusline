# Claude Statusline

A customized Claude [status line](https://code.claude.com/docs/en/statusline).

## Usage

1. Fork & clone this repo
1. Tweak `statusline-command.sh`
1. Open your Claude settings (`~\.claude\settings.json`)
1. Set `statusLine.command` to absolute path of this repo's `statusline-command.sh`

## Why?

Why not just use Claude's default `statusline-command.sh` in your ~/.claude folder?

Here are some reasons:

1. Github backup
1. Switching between machines
1. Tracking changes
1. Easier editing (open as separate project in your IDE, not some file from your user folder)
1. Sharing with others
